## Conceptos trabajados

- Definición de funciones.
- Parámetros y retorno.
- Separación de responsabilidades.
- Lectura de archivos.
- Procesamiento por bloques.
- Uso de funciones para evitar repetir lógica.

## Uso recomendado

Estudia cada ejercicio preguntando qué parte del problema merece convertirse en función. Una buena función no existe porque sí: existe porque encapsula una operación con nombre claro, entrada definida y salida esperada.
