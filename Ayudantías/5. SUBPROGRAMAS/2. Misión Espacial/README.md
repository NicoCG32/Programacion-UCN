# Misión Espacial

Ejercicio integrador de funciones, fórmulas y clasificación de datos.

## Contenido

| Archivo | Descripción |
| --- | --- |
| [Ej.py](<Ej.py>) | Archivo de trabajo del ejercicio. |
| [Mision.py](<Mision.py>) | Solución del ejercicio. |
| [exoplanetas.txt](<exoplanetas.txt>) | Archivo de entrada con datos de exoplanetas. |

## Conceptos trabajados

- Conversión de unidades.
- Funciones con fórmulas.
- Clasificación por rangos.
- Búsqueda de máximo y mínimo.
- Promedios condicionales.
- Construcción de informes por registro.

## Uso recomendado

Lee primero las fórmulas del enunciado y asígnales una función propia. En problemas con cálculos largos, nombrar bien las funciones ayuda a distinguir la matemática del procesamiento general del archivo.
