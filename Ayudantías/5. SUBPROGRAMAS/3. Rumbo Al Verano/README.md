Ejercicio integrador de subprogramas y cruce de archivos.

**Estado**: Implementado.

Consulta [EXPLICACIÓN.md](EXPLICACIÓN.md) para estudiar el cruce por fecha y sus costos.

## Contenido

| Archivo | Descripción |
| --- | --- |
| [Ej.py](<Ej.py>) | Archivo de trabajo del ejercicio. |
| [Rumbo al Verano.py](<Rumbo al Verano.py>) | Solución del ejercicio. |
| [dieta.txt](<dieta.txt>) | Archivo de entrada con comidas y calorías. |
| [entrenamiento.txt](<entrenamiento.txt>) | Archivo de entrada con entrenamientos y gasto calórico. |

## Conceptos trabajados

- Funciones de búsqueda por fecha.
- Conversión de tiempo `MM:SS` a segundos.
- Lectura de dos archivos relacionados.
- Cálculo de calorías consumidas y quemadas.
- Evaluación de metas.
- Manejo de casos sin dieta o sin entrenamiento.

## Uso recomendado

Este ejercicio es útil para estudiar cómo dividir un problema en módulos: una función calcula tiempo, otra lee entrenamientos, otra lee dieta y el programa principal coordina el análisis por fecha.
