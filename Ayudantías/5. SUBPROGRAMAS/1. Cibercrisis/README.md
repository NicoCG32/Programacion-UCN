# Cibercrisis

Ejercicio integrador de subprogramas y lectura de archivos.

## Contenido

| Archivo | Descripción |
| --- | --- |
| [Ej.py](<Ej.py>) | Archivo de trabajo del ejercicio. |
| [cibercrisis.py](<cibercrisis.py>) | Solución del ejercicio. |
| [ataques.txt](<ataques.txt>) | Archivo de entrada con continentes, países y ataques. |

## Conceptos trabajados

- Funciones para clasificar información.
- Lectura de archivo por bloques.
- Contadores por categoría.
- Cálculo de costos.
- Búsqueda de mayor promedio.
- Reportes por continente y país.

## Uso recomendado

Antes de revisar la solución, separa el problema en funciones: detectar malware, calcular costo, determinar ataque más recurrente y procesar bloques del archivo. Esa separación vuelve más legible un problema que, escrito todo junto, se vuelve difícil de seguir.
