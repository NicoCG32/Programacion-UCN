# Ayudantías de Programación

Material de ayudantías elaborado por Pablo Guzmán para apoyar el curso **Programación**.

Estas carpetas reúnen guías, ejercicios y soluciones en Python. El foco está en practicar las ideas centrales del curso con problemas pequeños, trazables y cercanos al formato de evaluación.

## Uso recomendado

1. Lee primero la guía o el enunciado en PDF.
2. Intenta resolver el ejercicio antes de abrir la solución.
3. Ejecuta los `.py` desde Spyder, cuidando que los `.txt` estén en la misma carpeta de trabajo.
4. Revisa el código buscando la lógica central: contador, acumulador, búsqueda, recorrido, función o lista.

Estas ayudantías deben estudiarse como práctica de razonamiento, no como recetas para copiar.
