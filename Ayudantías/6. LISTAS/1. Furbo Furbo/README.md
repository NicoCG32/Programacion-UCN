# Furbo Furbo

Ejercicio de listas paralelas aplicado a una liga deportiva.

**Estado**: Implementado.

Consulta [EXPLICACIÓN.md](EXPLICACIÓN.md) para revisar invariantes, búsquedas y ordenamiento.

## Contenido

| Archivo | Descripción |
| --- | --- |
| [Ej1 Ayudantía Listas.py](<Ej1 Ayudantía Listas.py>) | Solución del ejercicio. |
| [equipos.txt](<equipos.txt>) | Datos iniciales de equipos, partidos ganados, empatados, perdidos y goles. |
| [partidos.txt](<partidos.txt>) | Resultados de partidos que actualizan la tabla. |

## Conceptos trabajados

- Listas paralelas.
- Búsqueda de un equipo por nombre.
- Actualización de estadísticas.
- Cálculo de puntaje.
- Diferencia de goles.
- Desempate por criterios sucesivos.
- Ordenamiento de tabla manteniendo sincronizadas las listas.

## Uso recomendado

Ejecuta el programa con `equipos.txt` y `partidos.txt` en la misma carpeta. El punto importante es observar cómo un mismo índice representa al mismo equipo en todas las listas.
