## Conceptos trabajados

- Variables booleanas.
- Comparadores.
- Condiciones simples y compuestas.
- Uso de `if`, `elif` y `else`.
- Diferencia entre asignación y comparación.
- Corrección de errores sintácticos y lógicos.

## Uso recomendado

Primero revisa el PDF y luego abre el código incompleto. La idea es detectar por qué el programa falla, corregirlo y explicar qué condición permite autorizar o rechazar el acceso.
